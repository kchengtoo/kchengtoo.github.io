Circle For Roommates.

Circle For Roommates is an app that provides a roommate referral service, which helps users match with other users as potential roommates. Joining the team as a UI/UX designer, I helped conducting research and designing the interface to develop their new Circle Map feature where users can find roommates through neighborhood preferences.

2019 - UI/UX Design
Illustrator, Photoshop, After Effects, Sketch, InVision, Anima

Problem

Finding the right roommate has never been an easy task, especially when you are roomming with strangers, and Circle for Roommates has been using the "card swipe" model to help users relieve some of that frustration. However, there are users who are looking for roommates in a neighborhood first, before trying to figure if they are the perfect roommates. This points to the direction that the app will need to provide an alternative method to help find roommates through neighborhood preferences.

Could finding roommates be any easier?

Process

Circle for Roommates is going a step further with their roommate referral service by adding a new feature called Circle Maps. When I joined the team, some work of this feature had been done by previous UX designers. My job is to help create prototypes that show how users' neighborhood preferences are displayed on the map and their profiles, and then go through usability testing to improve the design.

Designing location preferences for Circle Maps within five weeks.

Research

We started with screening questions to filter out users who do not find locations before potential roommates in order to determine how neighborhood preferences should be displayed. We were able to find 8 participants to move forward with our user interviews.

Insight

Once we finished conducting the interviews, we began to debrief and determine how users can find the neigborhood they prefer to live in and let other users know. Here are the highlights:

  • Users should be able to select multiple neighborhoods that they want to live in
  • Display users' neighborhood preferences on their profile
  • For privacy purposes, show the general area rather than the specific location

Design

Since many features, like the Circle Map, had already been designed by previous designers, we have to design the neighborhood preference feature with the same design style of the existing features. The previous designers had been working with the Circle Map of Los Angeles, so we did the same thing for our design.

Circle Map of Los Angeles designed by Zoey Chung.

Using the insights that we obtained from our user interviews, we began to establish the necessary tasks to create the neighborhood preference feature:

  • Design an interface where users can select the neighborhoods that are in the city they are at
  • Create neighborhood preference tags on users' profiles
  • Design the pins that shows users' neigborhood preferences on the Circle Map

Neighborhood Preference Design

We designed the interface in a way that displays a simple map and lists out the neighborhoods in LA. When users select a neighborhood from the list, it sends visual feedbacks by highlighting their choices on the list and the map. Once users made their selection, their profile settings will display the number of neighborhoods that the users have selected.

User flow of the Neighborhood Preference feature.

Neighborhood Tag Design

We wanted to make sure that users can identify other users on Circle for Roommates who match their neighborhood preferences, so we included that information by adding a new tag called "Neighborhood Preferences" on users' profiles to indicate their choice of neighborhoods.

Neighborhood tags on users' profile. Users' Profile wireframe by Zoey Chung.

Circle Map Pin Design

On the Circle Map, we designed pins that will pinpoint on the map to indicate users' neightborhood preferences. Since the pins show which neighborhoods they want to live in, we created a zone around the pins to emphasize the neighborhood.

Circle Map pins on the map of LA. Circle Map by Zoey Chung.

Prototype

Once the neighborhood perference, tag on profile and Circle Map pin designs had been decided, it was time to combine them into one cohesive prototype to show the user flow of this new feature.

User flow of the Neighboorhood Preference feature.

Conclusion

I had a great fortune joining the Circle for Roommates team to work on an app that will help thousands of users to find their perfect roommates. As one of the earliest UI/UX design projects, I learned a lot about what this profession is like and prompted my interest in creating engaging user experience. If I can improve one aspect of this project, it would be to better at conducting the interviews to understand the users.
